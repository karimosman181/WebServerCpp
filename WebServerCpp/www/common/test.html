Feature rich

Built with C++ at its core, WebServerCpp offers unmatched performance, delivering fast responses with minimal overhead. Combined with a reactive JavaScript frontend, it provides a seamless development experience.

WebServerCpp follows a modular structure, allowing developers to extend the server with new functionalities easily.

Leverage Tailwind CSS for utility-first design without the bloat of large frameworks. Add reactivity to your web pages with Alpine.js, a minimal and lightweight JavaScript framework that enhances user interaction with minimal effort.

  • Modular Design
  • Thoughtful folder structure
  • Routing and controllers
  • JSON support
  • Templating
  • And much, much more.
